# Atribuição

     Operador        Exemplo         Equivalente     

        =               x = y             x = y      
        +=              x += y            x = x + y
        *=              x *= y            x = x * y
        /=              x /= y            x = x / y
        %=              x %= y            x = x % y


    Tipos de Operadores

    Operador                            Descrição

        +                               Adição
        -                               Subtração
        *                               Multiplicação
        **                              Exponencial
        /                               Divisão
        %                               Módulo
        ++                              Incrementar
        --                              Decrementar

    Operadores de Comparação

    Operador                           Descrição

        ==                              Igual a
        ===                             Mesmo valor e mesmo tipo
        !=                              Diferente
        !==                             Valor e tipos Diferentes
        >                               Maior que
        <                               Menor que
        >=                              Maior ou igual
        <=                              Menor ou igual

    Lógica 

    Operador                            Descrição

        &&                              "é" lógico
        ||                              "ou" lógico
        !                               "não" lógico

    Condicional (ternário)
        
        ----------- False ------------------
        |    ------ true ---               |
        |    |             |               |   
    (codição)    ?     expressão1  :  expresão2

    Comparação ao If e else

    if (condição)
       expressão1;
    else
       expressão2;
